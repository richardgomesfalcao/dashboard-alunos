# Dashboard de Alunos

Sistema de controle de alunos com login, cadastro, edição e exclusão.

## Tecnologias
- React
- Firebase (Auth + Firestore)
- Material UI

## Funcionalidades
- Login com autenticação
- CRUD de alunos
- Navegação entre páginas

## Próximos passos
